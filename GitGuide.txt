1. Клонирование репозитория: открываем git bash в папке где хотим проявить репозиторий.
команда: git clone *ссылка*
2. Закрываем этот гитбаш, заходим в папку репозитория, открываем там git bash
3. Прописываем команду git status, чтобы удостовериться что мы в репозитории.
4. Создаем ветку git checkout -b new_branch (git checkout команда переноса на другую ветку, параметр -b означает что мы создаем новую ветку)
5. Находимся в своей ветке (можно проверить командой git status)
6. Делаем необоходимые изменения.
7. git add . - для добавления ВСЕХ изменений в коммит
8. git commit -m "commit message" - для добавления коммита
9. перед !первым! пушем нужно связать локальный и удаленные репозитории !(делать только первый раз!)!
git remote add origin *ссылка*
10. git push origin <имя ветки>
